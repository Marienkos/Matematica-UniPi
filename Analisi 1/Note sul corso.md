Questo corso presenta un sistema di color coding per facilitare la classificazione mentale delle dimostrazioni:
- ~={red}ROSSO=~ per l'assurdo
- ~={orange}ARANCIONE=~ se si basano fortemente sul definitivamente
- ~={yellow}GIALLO=~ per l'induzione
- ~={green}VERDE=~ per il "cherry-picking" di $\varepsilon$ o $\delta$

Il ~={cyan}CIANO=~ è utilizzato per le dimostrazioni non classificabili in una delle precedenti categoria, solitamente per uno dei seguenti motivi:
- La dimostrazione discende direttamente da una definizione
- È applicato un teorema precedentemente dimostrato

Per non dare fastidio, solo l'header DIMOSTRAZIONE sarà colorato.
