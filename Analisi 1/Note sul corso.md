Questo corso presenta un sistema di colorazione per facilitare la classificazione mentale delle dimostrazioni, pertanto è necessario il plugin Fast Text Color.

I colori sono:
- ~={pink}ROSA=~ per le dimostrazioni banalmente derivate da un teorema o una definizione 
- ~={red}ROSSO=~ per l'assurdo
- ~={orange}ARANCIONE=~ per le dimostrazioni che si basano fortemente sul definitivamente
- ~={yellow}GIALLO=~ per l'induzione
- ~={green}VERDE=~ per il "cherry-picking" di $\varepsilon$, $\delta$ o successioni
- ~={cyan}CIANO=~ per le dimostrazioni complesse, multi-categoria o inclassificabili

Per non dare fastidio, solo l'header DIMOSTRAZIONE sarà colorato.
