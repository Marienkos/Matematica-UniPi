Il modo migliore per leggere questi appunti è utilizzare Obsidian non sperando in una qualche forma di coerenza stilistica.

L'idea generale è che ogni riga costituisce una nozione atomica Zettelkasten-like, più righe sono unite in blocchi preceduti da header di primo livello.

I file esistono solo per una maggiore comprensione e categorizzazione, fungendo da macro-temi.

Non ho veramente un'idea di cosa io stia facendo, spero solo di starlo facendo bene :)
